# 프로젝트 진행 보고 - README

본 폴더는 AI 고민상담 트리 구조 대화 시스템 프로젝트의 주간 진행 보고서를 보관합니다.

## 보고서 형식

### 파일명 규칙
`주간_진행보고_YYYYMMDD.md`

예시: `주간_진행보고_20251105.md`

### 보고서 내용
각 주간 보고서는 다음 섹션을 포함합니다:

1. **주간 요약**
   - 완료된 Phase
   - 진행중인 Phase
   - 다음 주 계획

2. **마일스톤 진척도**
   - 각 마일스톤의 예상 vs 실제 진행률

3. **이슈 요약**
   - 금주 발견된 이슈
   - 해결된 이슈
   - 미해결 이슈 현황

4. **크리티컬 패스 상태**
   - 크리티컬 패스상의 Phase 진행률
   - 지연 또는 가속 요인

5. **담당자 피드백**
   - 각 담당자의 주요 의견
   - 블로커 및 요청 사항

6. **다음 주 액션 아이템**
   - PM이 조치할 사항
   - 담당자별 우선순위 작업

---

## 보고서 작성 주기

- **매주 금요일**: 해당 주의 진행 보고서 작성
- **긴급 상황**: 크리티컬 이슈 발생 시 임시 보고서 작성 가능

---

## 보고 대상

- 프로젝트 오너 (필요 시)
- 기술 총괄자
- 담당자 전체 (투명성 유지)
