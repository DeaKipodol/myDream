다음 기능은 프로토타입 이후 단계에서 구현:
- 체크포인트/스냅샷 저장

- 보존 스레드 관리
- AI 통합
- 실제 데이터베이스 (MongoDB/DynamoDB)
- 인증/권한
- 배포 파이프라인

## 성공 기준

프로토타입 완성 기준:
1. 사용자가 노드를 클릭하면 경로가 전환되는 것을 시각적으로 확인할 수 있다
2. 브레드크럼에서 현재 경로를 확인할 수 있다
3. 대화 패널에서 활성 경로의 메시지만 표시된다
4. A→B→D 상태에서 C 클릭 시 A→C로 전환되는 것을 확인할 수 있다
5. A→B→D 상태에서 F 클릭 시 A→B→F로 전환되는 것을 확인할 수 있다
